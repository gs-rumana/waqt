## [0.0.2] - 2026-04-20

### Added
- **AI Tool Detection**: Automatically detects completions mapped to extensions like GitHub Copilot, Tabnine, Continue, Supermaven, Claude Dev, etc.
- **CLI Detection**: Advanced tracking capability to detect AI completions via command-line terminals (e.g. Claude Code, Gemini CLI, Aider).
- **Editing Fallback System**: Introduces advanced fallback mechanisms measuring heavy "pastes" unassociated with active AI assistance to help filter ChatBot/Web responses. 
- **Debug Tracking**: Automatically isolates and records hours spent running VS Code debug sessions. 
- **Multi-Editor Support**: Now displays accurate tracking details inside VSCodium forks (Cursor, Antigravity, Windsurf, Positron, VSCodium).
- **Responsive Dashboard**: Theme-native GUI built for light, dark, and high-contrast setups that wraps cleanly at small widths.
